Private Libraries

The plugin is opt-in: each user can turn on "Restrict my library" from the home-screen widget, after which they only see the media they requested through Jellyseerr or added from that widget. Users who never enable it keep seeing the full library.

Prefix for the per-user tag written onto granted items (e.g. jpl).
Off (default): the plugin is opt-in — each user is only restricted once they enable it from the widget. On: every user is restricted automatically and starts seeing nothing until granted something.

Paste this same value into the Jellyseerr webhook payload's secret field. Leave empty to disable webhook processing.

Jellyseerr webhook setup

In Jellyseerr → Settings → Notifications → Webhook, set the Webhook URL to <server>/PrivateLibraries/Webhook and use this JSON payload template: 

{
  "notification_type": "{{notification_type}}",
  "secret": "<your secret>",
  "media": {
    "media_type": "{{media_type}}",
    "tmdbId": "{{media_tmdbid}}",
    "tvdbId": "{{media_tvdbid}}"
  },
  "request": {
    "username": "{{requestedBy_username}}",
    "email": "{{requestedBy_email}}"
  }
}

Hidden media

Hide selected movies or series from every user except administrators. Hidden titles disappear from all non-admin libraries immediately, regardless of each user's own restriction setting. Changes take effect right away; the periodic reconcile task keeps them enforced.
